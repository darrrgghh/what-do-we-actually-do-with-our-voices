# Data Dictionary

## Stage 1 (interviews)

### `data/processed/stage1_segments_18.csv`

- `Vocalist` - participant code (`A`-`R`)
- `Gender` - self-reported gender (harmonized labels)
- `Age` - age in years
- `InterviewDate` - interview date (MM/DD/YYYY)
- `Experience_months` - reported extreme-vocal experience in months
- `Theme` - high-level thematic block
- `Quote` - coded text segment
- `Codes` - semicolon-separated code list
- `PrimaryCode` - dominant code for the segment
- `CodingSource` - `manual_existing` or `assisted_stage1_completion`

### `data/codebooks/codebook_v2_stage1_stage2.csv`

Extended codebook with original stage-1 schema and stage-2 additions.

## Stage 2 (Qualtrics)

### `data/processed/qualtrics_deidentified.csv`

Full de-identified export used for reproducible analysis.

### `data/processed/stage2_text_segments.csv`

Open-ended Qualtrics responses split into text units for coding.

- `RespondentID` - generated anonymous ID
- `QuestionCode` - survey question/field code
- `Theme` - mapped thematic group
- `Quote` - response segment
- `Codes` - assigned code list
- `PrimaryCode` - dominant code

## Reports and outputs

- `results/reports/sample_overview.csv` - participant and response counts
- `results/tables/` - manuscript-ready tables
- `results/tables/top15_bands.csv` - counts of `BAND_INFLUENCE(...)` name tokens in Stage 1 `Codes` (balanced parenthesis parsing)
- `results/tables/top15_vocalists.csv` - counts of `VOCALIST_INFLUENCE(...)` name tokens in Stage 1 `Codes`
- `manuscript/figures/` - PNG figures for the LaTeX manuscript (bar chart, Q44 heatmap, word clouds; regenerated by `scripts/pipeline/build_figures.py` and `build_theme_wordclouds.py`)
